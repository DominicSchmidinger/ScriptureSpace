# Scripture Space

Scripture Space is a web app for believers in Bible study groups. It helps organize studies while also being a peaceful place for quiet time with God. The platform combines community, reflection, and spiritual growth tools in one space.

## Features

- Daily random Bible verse
- Bible study organization tools
- Categorized discussion questions
  - Essential conflicts
  - Casual discussions
  - Deep faith topics
- Shared prayers for group prayer time
- 24-hour testimonies and experiences with God
- Bible reading tracker
- Daily prayer tracker with reminders
- Weekly recap of verses and study topics
- Reading challenge discussions and chapter reflections

## Goal

The goal of Scripture Space is to help believers stay connected with God and with each other while building consistency in prayer, Bible reading, and fellowship.
